Trees and Their Details

Mango Tree

Scientific Name: Mangifera indica

Height: 10–40 meters

Uses: Fruits, shade, wood, cultural importance

Banyan Tree

Scientific Name: Ficus benghalensis

Height: 20–30 meters

Uses: Shade, cultural and religious significance

Coconut Tree

Scientific Name: Cocos nucifera

Height: 15–30 meters

Uses: Coconut fruit, oil, coir, timber

Neem Tree

Scientific Name: Azadirachta indica

Height: 15–20 meters

Uses: Medicinal purposes, shade, pest control

Peepal Tree

Scientific Name: Ficus religiosa

Height: 20–30 meters

Uses: Religious importance, oxygen supplier, shade

Teak Tree

Scientific Name: Tectona grandis

Height: 30–40 meters

Uses: Strong timber, furniture, construction
