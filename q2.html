Notes on HTML Lists

1.Ordered List

An ordered list displays items in a specific sequence (numbers or letters).

Type Attribute:

  • type="1" → Numbers
  • type="A" → Uppercase Letters
  • type="a" → Lowercase Letters
  • type="I" → Uppercase Roman
  • type="i" → Lowercase Roman

Example:

  1. HTML
  2. CSS
  3. JavaScript

2. Unordered List (<ul>)

An unordered list displays items with bullets or symbols.

Type Attribute:

  • type="disc"
  • type="circle"
  • type="square"

Example:

  • Apple
  • Mango
  • Orange

3. Description List (<dl>)

A description list is used for terms and their definitions.

Tags Used:

  • <dl> → Defines description list
  • <dt> → Defines term
  • <dd> → Defines description

Example:

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
