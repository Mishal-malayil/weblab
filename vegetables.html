Vegetables and Their Details

Carrot

Scientific Name: Daucus carota

Color: Orange (also purple, yellow, red)

Nutrients: Rich in Vitamin A (beta-carotene)

Uses: Salads, soups, juice, cooking

Tomato

Scientific Name: Solanum lycopersicum

Color: Red (sometimes yellow or green)

Nutrients: Vitamin C, antioxidants (lycopene)

Uses: Curries, sauces, salads, juice

Potato

Scientific Name: Solanum tuberosum

Color: Brown skin, white/yellow inside

Nutrients: Carbohydrates, potassium, Vitamin C

Uses: Chips, fries, curry, mashed potato

Onion

Scientific Name: Allium cepa

Color: Red, white, yellow

Nutrients: Vitamin C, antioxidants

Uses: Curry base, salads, pickles
