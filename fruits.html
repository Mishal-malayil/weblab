Fruits and Their Details

Apple

Scientific Name: Malus domestica

Color: Red, Green, Yellow

Nutrients: Vitamin C, fiber, antioxidants

Uses: Eaten fresh, juice, desserts, salads

Banana

Scientific Name: Musa

Color: Yellow (when ripe), Green (unripe)

Nutrients: Potassium, Vitamin B6, fiber

Uses: Fresh fruit, smoothies, snacks

Mango

Scientific Name: Mangifera indica

Color: Yellow, Orange, Green (varieties)

Nutrients: Vitamin A, Vitamin C, antioxidants

Uses: Juices, desserts, pickles, eaten fresh

Orange

Scientific Name: Citrus sinensis

Color: Orange

Nutrients: Vitamin C, folate, fiber

Uses: Fresh fruit, juices, desserts

Grapes

Scientific Name: Vitis vinifera

Color: Green, Red, Purple

Nutrients: Antioxidants, Vitamin K, Vitamin C

Uses: Fresh fruit, wine, raisins, juice

Pineapple

Scientific Name: Ananas comosus

Color: Yellow flesh, brown-green outside

Nutrients: Vitamin C, manganese, fiber

Uses: Juices, desserts, salads, cooking
